# 主链路接线说明：错题模块

## 主线更新

原主线：
真题导入 → 题库入库 → 刷题作答 → 自动判分 → 错题沉淀 → 复盘训练

现主线更新为：
真题导入 → 题库入库 → 刷题作答 → 自动判分 → `error_analysis` 式错题管理 → 复盘训练

---

## 为什么这么改
因为错题不是训练末端的附属品，而是训练系统真正发生“诊断与再训练”的地方。

---

## 代码层后续接入位置

### 1. 练习页
来源：
- `/practice`

动作：
- 提交答案
- 触发判分
- 若错误，直接写入错题管理模块所需数据

### 2. 结果页
来源：
- `/practice`
- `/practice/result`

动作：
- 能跳转到错题管理模块
- 能从错题管理模块回到再练链路

### 3. 错题模块
目标：
- 作为正式工作台
- 而不是简单列表页

### 4. 复盘训练
来源：
- 错题模块中的聚合、诊断、再练动作
