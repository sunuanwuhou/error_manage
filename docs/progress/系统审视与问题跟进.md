## 问题一：用户坚持的动力来自"看到进步"，而非通知型激励

**发现时间**：2026-03-18  
**状态**：❌ 未解决  
**优先级**：🔴 高

### 问题描述

所有算法都建立在用户每天70道题的基础上。但现有留存设计（Day7里程碑、打卡徽章、streak）都是**通知型**激励，不是**内驱型**激励。

用户真正需要回答的问题是：**我今天练了，考试会不会涨分？**

现在进度页的"存量分=18分"没有情感冲击力。缺少的是预测分的情感化展示：

```
📈 按当前节奏，考试当天预计：
   保守估计：72分（存量底线）
   乐观估计：81分（含考前激活）
   目标：85分  差距：4分
   
   需要：还要存量化 3 个高频考点
   建议：重点攻判断推理（离门槛最近）
```

这一张卡片，比所有留存机制加起来都有效。

### 需要做什么

- [ ] 进度页新增"考试预测"核心卡片
- [ ] 显示保守/乐观双预测分，而非单一数字
- [ ] 显示"差距来自哪里"（具体到考点）
- [ ] 显示"今天练了涨了多少"（每日存量增量）

### 验证方法

用系统备考 7 天，每天能否在进度页清楚看到"今天的练习让预测分涨了 X 分"。  
若能，则认为解决；若每天打开感觉"不知道自己在进步"，则未解决。

## 问题二：AI 诊断冷启动质量差（知识库未预热）

**发现时间**：2026-03-18  
**状态**：❌ 未解决  
**优先级**：🔴 高

### 问题描述

现在 AI 诊断基于 RAG 检索知识库，但知识库是空的。冷启动状态下，AI 诊断质量和通用 Claude 一样，没有行测专属积累。

```
现在的情况：
  用户第1道错题 → AI诊断 → 知识库空 → 通用回答
  
应该的情况：
  提前预热：真题+解析 → 知识库500条解法模式
  用户第1道错题 → AI诊断 → 知识库命中 → 行测专属回答
```

### 需要做什么

- [ ] 收集历年真题（国考/省考，3-5套）→ 批量导入
- [ ] 收集高质量解析（粉笔/华图/官方）→ 通过"好题喂给AI"导入知识库
- [ ] 运行分析服务第一次（analysis-worker）
- [ ] 验证知识库条目数量 ≥ 100 条后，对比诊断质量

### 验证方法

取同一道错题（充分必要条件相关），分别在知识库空和预热后各诊断一次，对比：
- 诊断是否提到了具体的行测方法论（如"翻译推理三步法"）
- 行动规则是否具体可执行（如"看到只有...才→必要条件→否前否后"）

const insight = await prisma.$queryRaw`
  SELECT "paramValueNew" FROM system_insights
  WHERE "userId" = ${userId}
    AND "paramKey" = 'interval_sequence'
    AND status = 'applied'
  ORDER BY "createdAt" DESC LIMIT 1
`
const interval = insight[0]
  ? JSON.parse(insight[0].paramValueNew)
  : INTERVAL_SEQUENCE  // 降级用默认值

## 问题五：导入后承接断裂（真题进不了练习队列）

**发现时间**：2026-03-18  
**状态**：✅ 已解决  
**解决时间**：2026-03-18

### 原问题

导入完成后三个承接断裂：

1. ExamTopicStats 为空 → 考点盲区检测无法工作
2. 导入的题不出现在今日任务 → 用户不知道去哪练
3. 没有质检 → 解析错误的题悄悄进了题库

### 解决方案

**Fix 1: ExamTopicStats 自动填充**
导入完成后按题型统计频率，自动 upsert ExamTopicStats，考点盲区检测立即可用。

**Fix 2: PracticeRecord.isPending**
新增 `isPending` 字段。导入后的题写入 `isPending=true`，daily-tasks 优先取待练池，
首页"真题练习"区块立即显示待练数量。用户答完后改为 `isPending=false`。

**Fix 3: 导入质检**
每道题解析后评分（0-100），检测：题目过短/选项数量/答案格式/选项混入正文。
低于50分的题跳过，其余问题在导入完成页展示质检报告。

### 验证方法

1. 导入一套真题PDF
2. 打开考点盲区页面 → 应有数据（不再提示"暂无频率数据"）
3. 打开首页 → 真题练习区块显示"X道真题待练"
4. 导入完成页 → 显示考点分布和质检问题（如有）
